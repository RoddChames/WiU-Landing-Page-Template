Project Name: WorldSurf

Project Description

Over the last couple of weeks me and my fellow classmates, here at Full Sail University Online, have been learning design & coding techniques, which will give us the upper hand we need to compete in the "real world" of design/developers. This page is the landing page for my current Web Interface & Usability class. During this course we will: expand your theoretical knowledge of interface design, gain personal insight into your skills with web design, and finally, build upon your existing skill set.

I will be creating a web application for an organization named "World Surf", in which I will be designing static wireframes (.jpgs) using Adobe Illustrator. During this course I am expected to also learn new terminologies (Design Patterns) crucial for designing and delivering professional web applications to clients and investors.
